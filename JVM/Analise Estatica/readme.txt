Análise Estática realizada com a ferramenta CppCheck.
1) Crie uma pasta e coloque apenas os fontes .c (.h ficam de fora)
2) Instale o CppCheck
   baixe ele no site http://cppcheck.sourceforge.net/
3) Execute o programa.
4) Acesse o menu Check -> Files
5) Selecione todos os arquivos de uma só vez.
6) Acesse o meu File -> Print e a análise estará disponível em pdf.
7) Para gerar o arquivo XML acesse o meu File -> Save results to file...
