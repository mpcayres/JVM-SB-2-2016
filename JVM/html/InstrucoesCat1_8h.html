void jvm_fcmpg ( )

Compara dois floats retirados da pilha.

Se os dois numeros forem iguais o int32_t 0 eh acrescetado a pilha. Se o primeiro valor for maior que o segundo o int32_t 1 eh acrecentado a pilha. Se o segundo valor for maior que o primeiro o int32_t -1 eh acrestado na pilha
